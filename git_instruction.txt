1. Для того, чтобы сделать свою копию репозитория, нужно сделать

git clone git@github.com:vasalf/mamooth-world.git

2. Для того, чтобы обновить свою копию, нужно сделать

git fetch
git merge

3. Коммитить нужно все, вплоть до самых мелких изменений

git commit -a

Комментарии к коммиту желательно писать на басурманском английском языке.

4. Чтобы слить свою копию в основной репозиторий, нужно сделать

git push

5. Чтобы удалить файл из проекта, нужно сделать к нему

git rm <имя_файла>

6. Чтобы добавить файл в проект, нужно сделать к нему

git add <имя_файла>

7. Чтобы переименовать или переместить файл, нужно сделать

git mv <старое_имя> <новое_имя>

8. Чтобы посмотреть текущие изменения, нужно сделать

git status

9. Чтобы посмотреть последние коммиты, нужно сделать

git log
